## 🎯 第一件事：小红书"钓鱼帖"获客（Day 1-3）

### 为什么是小红书？

| 平台 | 起量速度 | 精准度 | 适合你的原因 |
|------|----------|--------|-------------|
| 公众号 | 慢（2-4周） | 高 | 需要积累，不适合一周变现 |
| 抖音 | 中等 | 低 | 泛流量多，转化难 |
| **小红书** | **快（1-3天）** | **高** | 搜索流量大，用户主动找解决方案 |

小红书的核心逻辑：用户主动搜"AI写作"、"公众号爆文"等关键词 → 你的帖子出现 → 用户私信你

### 具体执行（精确到每一步）

**Day 1 上午：注册+装修账号**
```
昵称：李可｜AI内容效率
简介：美业三次创业者｜用AI 10倍提升内容效率｜私信"工具"领取提示词模板
头像：你的真人照片
```

**Day 1 下午：发布第1篇"钓鱼帖"**

标题模板（直接用）：
```
做了3年公众号，终于发现AI写文章的正确姿势
```

内容结构（800字左右）：
```
【痛点开头】
很多人用AI写文章，结果全是AI味，读者一眼就看出来了。
我也踩过这个坑。

【转折】
后来我发现，问题不在AI，在于提示词。
用对提示词，AI写的东西和人写的没区别。

【干货（给1-2个真实有用的提示词）】
比如这个提示词：
"请用口语化的方式，像和朋友聊天一样，写一篇关于XX的文章..."
（把你P9口语化大师的核心提示词给一个）

【钩子】
我整理了50个这样的提示词模板，涵盖公众号/小红书/短视频文案。
想要的私信我"工具"，免费发你。
```

**Day 2-3：每天发2篇类似的帖子**

选题池（都是用户会搜的关键词）：
1. 公众号爆文标题怎么写？我用AI 5分钟生成50个
2. AI写的文章太假？这3个技巧让它像人写的
3. 实体店老板怎么用AI写获客文案
4. 我用AI一天写10篇公众号，方法分享
5. 公众号没灵感？这个AI工具帮我解决了

### 预期结果

|------|----------|----------|

| 总曝光 | 1000-3000 | 5000-10000 |

## 🎯 第三件事：设计一个99元的"小产品"快速成交（Day 3-5）

### 为什么是99元？

| 价格 | 优劣势 |
|------|--------|
| 9.9元 | 门槛低但难筛选付费意愿强的人 |
| **99元** | **足够低让人尝试，足够高让人认真** |
| 199元+ | 需要更多信任，一周内难成交 |

### 产品设计（直接用）

**产品名：「AI内容效率急救包」**

**交付物：**
```
1. 50个精选提示词模板（PDF）
   - 10个公众号爆文标题公式
   - 10个口语化改写提示词
   - 10个短视频脚本模板
   - 10个小红书爆款结构
   - 10个实体店获客文案

2. 1个30分钟视频教程
   - 演示如何用这些提示词
   - 展示你的工作流程

3. 1次微信答疑（15分钟）
   - 针对他的具体问题
```

**定价策略：**
```
原价：199元
本周尝鲜价：99元
（仅限前10名）
```

### 成交话术（私聊用）

当对方表示感兴趣后：
```
我最近整理了一个「AI内容效率急救包」，包含50个我自己在用的提示词模板。

不是网上那种复制粘贴的，是我做了216个智能体、踩了90万坑之后，筛选出来最有效的。

本来定价199，这周刚做出来想找几个人测试，99元包含视频教程+1次答疑。

你有兴趣的话，我今天就给你开通？
```

## 成功关键：每个环节的具体话术

### 小红书评论区互动话术
```
当别人问："这个方法真的有用吗？"
→ "我自己用了3个月了，效果比我想象的好。你有具体问题可以私信我"

当别人问："能分享一下完整版吗？"
→ "私信我发你，评论区放不下"
```

### 微信添加后第一句话
```
"你好，我是李可，在小红书/群里看到你对AI内容效率感兴趣，
我这边整理了一份提示词模板，先发你看看合不合适"
（立刻发一个有价值的资料）
```

### 推产品前的铺垫
```
"你现在做内容最卡的点是什么？"
（先问问题，了解需求）

"这个问题其实我之前也遇到过，我后来是用XX方法解决的..."
（展示你能解决他的问题）

"我整理了一个完整的工具包，包含XX+XX+XX，你有兴趣了解一下吗？"
（软性推产品）
```
