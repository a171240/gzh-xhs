# 视觉风格库

配图提示词模板库，用于P4配图生成。

---

## 图片规格

| 图片类型 | 尺寸 | 用途 |
|---------|------|------|
| 封面图 | 900×383px | 公众号头条封面 |
| 章节配图 | 1080×1440px | 每个章节对应一张竖版配图 |

**配图数量**：1张封面 + N张章节配图（N = 文章章节数）
- 普通文章：通常5-6张章节配图
- 深度长文：通常8-10张章节配图

---

## 封面图构图规则（正方形优先法）

封面图在次条/分享时会被裁剪成正方形（383×383px，取中央区域）。
采用「正方形优先构图法」确保核心内容在任何场景下都完整：

```
┌─────────────────────────────────────────────────────────────────┐
│                       900×383px 完整画面                         │
│                                                                 │
│    ┌─────────┐  ┌─────────────────────────┐  ┌─────────┐       │
│    │         │  │                         │  │         │       │
│    │ 装饰区A │  │    ★ 核心区域 ★         │  │ 装饰区B │       │
│    │ 258px   │  │    383×383px            │  │ 258px   │       │
│    │         │  │    （居中正方形）        │  │         │       │
│    │ 可裁剪  │  │    所有核心内容在此     │  │ 可裁剪  │       │
│    └─────────┘  └─────────────────────────┘  └─────────┘       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**核心区域（383×383px）**：
- 完整中文标题（AI直接生成）
- 副标题/数据
- 主视觉元素
- 文字安全区：内缩至350×350px

**装饰区域（左右各258px）**：
- 仅放背景延伸元素
- 可被裁剪，不影响主题

**规则**：先完成核心区设计（正方形），再向两侧扩展装饰。裁剪后画面仍完整美观。

## 风格A：知识黑板报风格（IP内容工厂专用）

### 封面图提示词模板

```
【图片类型】微信公众号封面图
【尺寸】900×383像素

【⚠️ 核心构图规则 - 正方形优先法】
这张图会被裁剪成正方形（383×383px，取中央区域）。
所有核心视觉元素必须在中央383×383px正方形区域内。

【安全区域】
- 核心文字（主标题+数字亮点）必须集中在画面中央350×350px区域内
- 两侧258px为"可裁剪区"，只放装饰元素，不放重要文字
- 确保裁剪成正方形后，核心信息仍完整可读

【画面风格】
知识黑板报风格。深墨绿色黑板背景，带有真实黑板的细微划痕和粉笔灰尘纹理。

【核心区域内容】（383×383px中央正方形）
- 主标题：「[完整中文标题]」
  - 位置：核心区域正中央偏上
  - 字体：粗犷手写粉笔字，白色
  - 大小：占核心区宽度50-60%（约190-230px宽）
  - 效果：粉笔质感，边缘略带粉尘
- 副标题/核心数据：「[如：90%的人不知道]」
  - 位置：主标题下方
  - 字体：细一号粉笔字，淡黄色
  - 大小：主标题的40-50%
- 装饰元素（核心区内）：
  - 简单粉笔画框（虚线/波浪线）围绕标题
  - 小星号★或箭头→作为点缀

【装饰区域】（左右各258px，可被裁剪）
- 粉笔手绘的延伸装饰：
  - 左侧：书本简笔画、灯泡图标
  - 右侧：箭头、星号、波浪线
- 这些元素被裁剪后不影响主视觉

【色彩方案】
- 背景：深墨绿色(#2D4A3E)
- 主标题：白色
- 副标题：淡黄色(#F5E6A3)
- 装饰：浅灰/浅黄

【负面提示词】文字变形扭曲、文字模糊不清、文字偏离中央、重要内容在两侧、模糊、低分辨率、3D效果、真实照片
```

## 使用说明

1. **P4执行时**：根据账号名称自动选择对应风格
2. **核心原则**：账号配图风格强绑定，不可更改
3. **验证要求**：P4生成配图时检查风格是否匹配账号（P0级规则）
4. **模板使用**：直接复制对应风格的模板，填充文章内容即可
