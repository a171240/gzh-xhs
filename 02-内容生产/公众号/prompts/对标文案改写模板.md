# 对标文案改写模板

## 使用场景

当Brief中指定了"参考对标文案"时，使用本模板将对标文案的爆款逻辑转化为公众号内容。

---

## 改写原则

### 核心三原则

1. **保留核心逻辑，改写表达方式**
   - 复用对标文案的结构模板、情绪触发点、框架
   - 不照搬原文，用自己的语言重新表达
   - 融入李可IP人设和自有案例

2. **适配账号风格**
   - gongchang：强调系统性和框架性
   - ipgc：强调情感共鸣和人生阶段
   - zengzhang：强调数据和增长杠杆
   - shizhan：强调可操作性和避坑

3. **融入自有案例**
   - 用李可IP人设的真实经历替换对标案例
   - 用美容行业案例替换对标案例
   - 保持案例的"可信度"和"可复制性"

---

## 改写流程

### Step 1: 加载对标文案分析报告

从 `03-素材库/对标链接库/分析报告/` 读取对应的分析报告，提取：
- 爆款类型
- 结构模板
- 情绪触发点
- 可复用金句
- 核心观点
- 案例素材

### Step 2: 匹配账号风格

根据Brief中的目标账号，选择对应的改写方向：

**gongchang（系统方法论）**：
- 标题：强调框架、机制、对比、底层逻辑
- 开头：数据钩子 + 系统性问题
- 正文：框架展开 + 每步有逻辑
- 案例：用李可的"216个智能体工作流"、"291个提示词模板"等数据

**ipgc（情绪共鸣）**：
- 标题：强调人生阶段、关系张力、情绪反转
- 开头：场景钩子 + 情感共鸣
- 正文：故事展开 + 情绪触发
- 案例：用李可的"至暗时刻"、"两次创业崩盘"等真实经历

**zengzhang（数据增长）**：
- 标题：强调指标语言、样本洞察、增长杠杆
- 开头：数据钩子 + 增长案例
- 正文：数据支撑 + 增长逻辑
- 案例：用李可的"半年拓客6000+"、"1000+新客只沉淀60个"等数据

**shizhan（轻实操）**：
- 标题：强调避坑、清单、可马上执行
- 开头：避坑钩子 + 实操场景
- 正文：清单展开 + 每步可执行
- 案例：用李可的"两次创业踩坑90万"、"门店增长四步法"等实操经验

### Step 3: 替换案例

**对标案例 → 自有案例映射表**：

| 对标案例类型 | 自有案例替换 |
|------------|------------|
| 年入七位数/百万 | 李可：半年拓客6000+（抖音同城早期） |
| 从零开始/低成本起步 | 李可：16岁辍学做厨师，第一天地推不敢开口 |
| 踩坑/失败经历 | 李可：两次创业崩盘，累计亏损90万 |
| 转型/重启 | 李可：第三次创业（2025年10月10日） |
| 技能产品化 | 李可：216个AI智能体工作流，291个提示词模板 |
| 系统化方法 | 李可：门店增长四步法（定位→内容→转化→复盘） |

**美容行业案例库**：
- 1000+新客只沉淀60个，转化率<6%
- 客单价偏低（平均约80+）
- 转化环节失控（店长频繁变动）
- 扩张过早（未稳一店即开二店）

### Step 4: 生成四账号内容

按照对标文案的结构模板，生成四账号差异化内容：
- 保留对标文案的"结构骨架"
- 替换为自有案例和表达方式
- 适配各账号的标题公式和语言风格

## 改写示例

### 示例1：普通人逆袭三步法 → 四账号改写

**对标文案结构**：
```
开头：价值承诺钩子 + 真实案例背书
第一步：像素级复制（案例：李公子年入七位数）
第二步：像小强一样顽强（案例：三三厕所直播）
第三步：构建资产（案例：聂风年营业额1500万）
结尾：总结型 + 祝福型
```

**gongchang改写**：
```
标题：普通人做内容的三步法：从像素级复制到系统化
开头：我观察了200+做内容的朋友，发现80%的人都卡在第一步
第一步：像素级复制（案例：我搭建了216个AI智能体工作流）
第二步：暴力时长（案例：我半年拓客6000+的增长案例）
第三步：构建资产（案例：我的291个提示词模板）
结尾：这三步不是捷径，是系统
```

**ipgc改写**：
```
标题：30岁，两次创业崩盘，我用三步法重新出发
开头：2025年7月，爷爷去世后的高铁上，我和妻子爆发了激烈争吵
第一步：像素级复制（案例：16岁辍学做厨师，看到老师傅的样子，我决定转行）
第二步：暴力时长（案例：第一天地推不敢开口，站在街头两小时）
第三步：构建资产（案例：两次创业踩坑90万，我终于想明白这件事）
结尾：如果你也正在经历低谷，这三步或许能帮到你
```

**zengzhang改写**：
```
标题：从1000+新客只留60个，到半年拓客6000+，我做对了什么
开头：数据告诉你真相：转化率<6%的背后，是系统性问题
第一步：像素级复制（案例：我搭建了216个AI智能体工作流，效率提升10倍）
第二步：暴力时长（案例：半年拓客6000+，成功=命中率×出手次数×时长）
第三步：构建资产（案例：291个提示词模板，脱离时间限制）
结尾：这三步的增长杠杆，你一定要知道
```

**shizhan改写**：
```
标题：想做内容但不知道从哪开始？这三步最适合新手
开头：很多人做内容，第一步就踩坑了
第一步：像素级复制（避坑：不要一开始就想创新）
第二步：暴力时长（避坑：不要等完美了再开始）
第三步：构建资产（避坑：不要只打工不构建资产）
结尾：这三步，学完马上就能用
```

## 改写检查清单

### 必须检查的5个点

- [ ] **结构保留**：是否保留了对标文案的结构模板？
- [ ] **案例替换**：是否用自有案例替换了对标案例？
- [ ] **风格适配**：是否适配了目标账号的语言风格？
- [ ] **金句复用**：是否复用了对标文案的高价值金句？
- [ ] **人设一致**：是否符合李可IP人设？

### 禁止事项

- ❌ 照搬对标文案的原文
- ❌ 使用对标文案的案例（必须替换为自有案例）
- ❌ 忽略账号差异化（四账号标题前4字不重复）
- ❌ 丢失对标文案的核心逻辑
- ❌ 过度改写导致失去爆款逻辑

---

## 使用说明

### 触发条件

当Brief中包含以下字段时，自动使用本模板：
```
- 参考对标文案：{对标文案标题或路径}
```

### 执行流程

1. 读取对标文案分析报告
2. 提取爆款逻辑（结构/情绪/金句/观点）
3. 根据目标账号选择改写方向
4. 替换案例为自有案例
5. 生成四账号差异化内容
6. 检查改写质量

### 输出格式

按照公众号批量生产的标准格式输出：
- 四账号标题（前4字不重复）
- 四账号正文（适配各账号风格）
- 标注"参考对标文案：{标题}"
