## 一、新版6张图结构

| 图序 | 内容定位 | 信息密度 | 字数限制 |
|------|---------|---------|---------|
| **P1封面** | 数据可视化信息图（承载2000字完整信息） | 极高（20-30个信息点） | 按模块分布 |
| **P2** | 模块1通俗展开（痛点场景） | 低 | ≤60字 |
| **P3** | 模块2通俗展开（原因机制） | 低 | ≤60字 |
| **P4** | 模块3通俗展开（方法步骤） | 低 | ≤60字 |
| **P5** | 模块4通俗展开（效果证据） | 低 | ≤30字 |
| **P6** | 固定CTA页 | - | 固定 |

## 二、统一视觉规范

### 基础规范
- **画幅比例**：竖版 **3:4**（1080×1440px）
- **风格定位**：数据可视化 / 专业报告感
- **P1留白**：30-40%（信息充实）
- **P2-P5留白**：50-60%（易读）
- **可读性**：字号要大，手机端清晰可读

### 色彩体系

| 元素 | 色值 | 用途 |
|------|------|------|
| 背景主色 | #F8F8F8 → #EEEEEE渐变 | 所有页面背景 |
| 主标题色 | #333333 | 标题文字 |
| 正文色 | #555555 | 正文文字 |
| 辅助文字 | #666666 | 说明文字 |

### 账号专属配色

| 账号 | 主色 | 辅助色 | 数据高亮 | 角标 |
|------|------|--------|----------|------|
| A号（转化） | #7C3AED | #A78BFA | 紫色渐变 | 「诊断」 |
| B号（交付） | #6B7280 | #9CA3AF | 深灰高亮 | 「SOP」 |
| C号（观点） | #374151 | #6B7280 | 黑色高亮 | 「观点」 |

### 通用负面提示词
```
文字变形扭曲、文字模糊不清、字体过小无法阅读、错别字、乱码、
人物照片、复杂背景图案、英文字母、水印、品牌logo、二维码、
密集文字堆砌、装饰过多、色彩杂乱、卡通风格、手绘风格
```

## 四、P2-P5 通俗展开页模板

### 设计原则
- 每页对应P1信息图的一个模块
- 用大白话解释该模块的核心要点
- 降低理解门槛，让小白也能看懂
- 视觉简洁，突出核心信息

---

**对应模块**：P1信息图的模块1（问题/痛点）
**文案要求**：页面标题 + 3个痛点场景（每个≤15字）

**Prompt模板**：
```

【尺寸】竖版3:4（1080×1440px）

【画面风格】
简洁商务信息卡片。背景：雾白色(#FAFAFA)渐变。
{点缀色}作为强调色。留白充足50-60%。

【核心内容】
- 页面标题：「{页面标题}」
  - 位置：画面顶部，距顶12%
  - 字体：现代无衬线粗体，深灰色(#333333)
  - 大小：占画面宽度55%

- 三个痛点卡片（纵向排列）：
  - 卡片1：「{痛点1}」
  - 卡片2：「{痛点2}」
  - 卡片3：「{痛点3}」

  - 卡片样式：白色卡片(#FFFFFF)，左侧{点缀色}竖线标记
  - 文字：中灰色(#555555)，左对齐

【装饰元素】
- 卡片左侧{点缀色}竖线装饰
```

## 七、使用说明

### 完整工作流程

1. **准备完整内容**：先生成2000字级别的结构化内容
2. **生成P1信息图**：使用`信息图模板库.md`中的模板
3. **生成P2-P5展开页**：使用本文件的模板，填充对应模块内容
4. **选用P6 CTA**：直接使用预制CTA图片
5. **后期处理**：P5页面贴入真实截图

### 文案占位符说明

| 占位符 | 含义 | 字数限制 |
|--------|------|----------|
| `{页面标题}` | 内页标题 | ≤12字 |
| `{痛点N}` | 痛点描述 | ≤15字 |
| `{原因N}` | 原因说明 | ≤12字 |
| `{说明N}` | 补充说明 | ≤15字 |
| `{步骤N}` | 流程步骤 | ≤8字 |
| `{截图说明N}` | 截图标注 | ≤10字 |
| `{点缀色}` | 账号主色 | 按账号选择 |

### 质检清单

#### P1信息图
- [ ] 信息点数量：20-30个
- [ ] 核心数据：2个以上大字号高亮
- [ ] 模块间关系线清晰
- [ ] 配色与账号匹配

#### P2-P5展开页
- [ ] 用大白话解释封面内容
- [ ] 每页字数≤60字
- [ ] 手机端清晰可读
- [ ] 视觉风格统一

#### P6 CTA
- [ ] 使用正确账号的固定模板
- [ ] 无微信号/电话/站外链接
