# 标题
搜“美容院做脸有用吗”的人，怕的通常不是没效果

# 口播正文
很多人搜“美容院做脸有用吗”，不是在做知识题。  
是明天要见人，脸色发闷发糙，想把状态拉回来一点；又怕重来一遍那种尴尬：你明明说了不要，对方还继续讲。  

这个词真正想问的，不是项目名，而是我能不能被尊重。  
在椿舍，我们先看你当天状态，再决定今天做到哪一步；过程中可以推荐，但你说不要，话题就停。研究人性不是为了拿捏你，是为了避开你的雷区。  
所以“有用”不只看当下干不干净，也看你做完有没有松一口气。你不用一边做脸，一边防着下一句。不扫兴，才是情绪价值。

# 置顶评论
如果你也在意“拒绝是不是安全的”，先看这三条：  
1. 项目中不硬推，推荐可拒绝。  
2. 做前先说清当天护理范围，不临时变调。  
3. 不加价、不缩水、不逼办卡。  

门店位置：吴江区江兴西路地铁站附近 / 吴江公园附近  
新客信息：以平台团购页实时展示为准

# 回复模板
- 问：会不会一直推销？  
  答：不会。项目进行中不聊办卡和加项，先把护理完整做完。
- 问：敏感状态能做吗？  
  答：会先看当天状态，再决定走温和舒缓还是轻清洁，不会硬上强度。
- 问：一次大概多久？  
  答：通常约 70 分钟，含基础清洁与放松环节。
