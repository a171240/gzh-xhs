## 公众号写作核心原则

### 原则一：开头3秒抓住注意力
```
❌ 错误：今天来聊聊AI写作这个话题...
✅ 正确：90%的人用AI写内容，都卡在同一个地方。

❌ 错误：在当今数字化时代，AI技术的发展...
✅ 正确：我以前写一条脚本要4小时，现在10分钟。
```

### 原则二：像说话一样写作
```
❌ 书面腔：综上所述，AI工具对于提升效率具有显著作用。
✅ 口语化：说白了，用对了AI，效率能提10倍不止。

❌ 书面腔：经过反复验证和实践检验...
✅ 口语化：我试了很多次，发现...
```

### 原则三：数据和案例支撑观点
```
❌ 空洞：AI能大大提升效率
✅ 具体：我用AI写脚本，从4小时压缩到10分钟，一周能产出50条

❌ 空洞：很多人都遇到过这个问题
✅ 具体：我观察了200+做内容的朋友，发现80%的人卡在同一个地方
```

### 原则四：段落节奏控制
- 每段不超过4行
- 重要观点单独成段
- 用空行制造呼吸感
- 短句+长句交替使用

### 原则五：AI痕迹规避
**绝对不能使用**：
- 「这不是X，是Y」（假装深刻句式）
- 「更扎心的是...」（套路化转折）
- 「你知道吗？」（假装互动）
- 「总的来说...」（总结癖）
- 「我当时就懵了」（标准化反应）
- 「不仅...而且...」（教科书句式）

### 原则六：排版规范
```
【小标题】使用emoji增加视觉层次
💡 第一步：明确你的定位
📊 数据告诉你真相

【重点标注】**加粗**突出关键信息

【列表格式】
1️⃣ 第一点...
2️⃣ 第二点...

【分隔线】使用 --- 分隔区块
```
