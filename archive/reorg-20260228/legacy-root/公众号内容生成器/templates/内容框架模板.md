# 内容框架模板库

5种普通内容框架 + 1种深度长文框架，用于P2内容生成。

---

## 使用说明

1. **根据选题类型选择框架**：P1阶段会自动判定内容类型
2. **字数分配严格遵守**：每个部分的字数范围不可随意更改
3. **框架可灵活调整**：根据实际内容可微调结构，但核心要素不可缺失
4. **验证点**：Step 4验证大纲是否符合框架字数分配（P2实时验证）

---

## 框架1：深度干货（2500-3000字）

### 适用场景
- 方法论输出
- SOP流程讲解
- 系统化知识分享
- 原理深度解析

### 框架结构

```
📍 痛点开场（100字）
   └─ 抛出用户最关心的问题
   └─ 制造紧迫感和共鸣

📍 问题分析（300字）
   └─ 为什么会有这个问题
   └─ 问题的本质是什么
   └─ 不解决的后果

📍 方法论展开（1500字）⭐ 核心部分
   └─ 要点1：观点+案例+数据+行动
      - 提出观点（100字）
      - 案例佐证（150字）
      - 数据支撑（100字）
      - 行动指南（150字）

   └─ 要点2：观点+案例+数据+行动
      - 同上结构（500字）

   └─ 要点3：观点+案例+数据+行动
      - 同上结构（500字）

📍 案例佐证（500字）
   └─ 真实案例详细展开
   └─ 验证方法论有效性

📍 行动建议（200字）
   └─ 3个立即可执行的步骤
   └─ 避免的误区

📍 CTA引导（100字）
```

### 字数检查清单
- [ ] 总字数：2500-3000字
- [ ] 痛点开场：~100字
- [ ] 问题分析：~300字
- [ ] 方法论：~1500字（3个要点均衡分配）
- [ ] 案例：~500字
- [ ] 行动建议：~200字
- [ ] CTA：~100字

### 推荐账号
- ✅ **IP内容工厂**（最适合）：系统方法论
- ✅ **IP增长引擎**（适合）：数据驱动型干货
- ⚠️ IP工厂：需要加入更多情感和故事
- ⚠️ 商业IP实战笔记：需要简化为轻松实用风格

   └─ 核心功能

   └─ 步骤1：[具体操作]（200字）
   └─ 步骤2：[具体操作]（200字）
   └─ 步骤3：[具体操作]（200字）
   └─ 步骤4：[具体操作]（200字）

   每步包含：
   - 截图说明

📍 使用技巧（200字）
   └─ 3-5个高级技巧
   └─ 提高效率的小窍门

📍 效果展示（100字）
   └─ 前后对比数据

- [ ] 场景引入：~100字
- [ ] 工具介绍：~200字
- [ ] 实操步骤：~800字（4步均衡分配）
- [ ] 使用技巧：~200字
- [ ] 效果展示：~100字
- [ ] CTA：~100字

### 推荐账号
- ✅ **IP增长引擎**（最适合）：工具测评、数据对比
- ✅ **商业IP实战笔记**（适合）：轻松实操教程

## 框架3：个人故事（2000-2500字）

### 适用场景
- 个人经历分享
- 踩坑教训
- 成长复盘
- 情感共鸣类

### 框架结构

```
📍 场景代入（150字）
   └─ 具体时间、地点、场景
   └─ 制造画面感
   └─ 勾起读者好奇

📍 困境描述（400字）
   └─ 当时面临什么问题
   └─ 内心的恐惧/迷茫/焦虑
   └─ 尝试过的失败方法
   └─ 用细节和情绪打动人

📍 转折过程（800字）⭐ 核心部分
   └─ 转折点：什么让你改变
   └─ 行动：具体做了什么
   └─ 过程：遇到的困难和坚持
   └─ 结果：最终达成的效果

📍 收获反思（500字）
   └─ 这段经历让你学到什么
   └─ 哪些认知被改变了
   └─ 给其他人的具体建议

📍 价值升华（250字）
   └─ 从个人经历提炼普适价值
   └─ 金句总结核心观点

📍 CTA引导（100字）
```

### 字数检查清单
- [ ] 总字数：2000-2500字
- [ ] 场景代入：~150字
- [ ] 困境描述：~400字
- [ ] 转折过程：~800字（最详细部分）
- [ ] 收获反思：~500字
- [ ] 价值升华：~250字
- [ ] CTA：~100字

### 推荐账号
- ✅ **IP工厂**（最适合）：情绪共鸣、真实经历
- ⚠️ IP内容工厂：不适合（偏方法论）
- ⚠️ IP增长引擎：不适合（偏数据）
- ✅ **商业IP实战笔记**（适合）：踩坑教训类故事

- 热点事件分析
- 趋势解读
- 新闻关联

### 框架结构

```

   └─ 简述热点事件
   └─ 为什么值得关注

   └─ 对普通人的影响是什么
   └─ 背后的本质是什么

📍 独特观点（600字）⭐ 核心部分
   └─ 你对这个热点的独特见解
   └─ 与主流观点的不同
   └─ 用案例和数据支撑观点

📍 方法论回扣（400字）
   └─ 从热点回到方法论
   └─ 给用户的具体建议
   └─ 如何应对/抓住机会

📍 CTA引导（100字）
```

### 字数检查清单
- [ ] 总字数：1500-2000字
- [ ] 热点切入：~150字
- [ ] 关联分析：~400字
- [ ] 独特观点：~600字
- [ ] 方法论回扣：~400字
- [ ] CTA：~100字

### 推荐账号
- ✅ **IP内容工厂**（适合）：深度分析、独特观点
- ✅ **IP增长引擎**（适合）：数据解读、趋势分析
- ⚠️ IP工厂：需要加入个人故事和情绪

- 回扣方法论，不能离题太远

## 框架6：深度长文（6000-10000字）

### 适用场景
- SEO长尾流量
- 建立专业权威
- 引流钩子/白皮书
- 系统性知识输出

### 框架结构

```
📍 引言/钩子（300-500字）
   └─ 核心观点预告
   └─ 为什么要读这篇文章
   └─ 读完能获得什么

📍 📑 目录（自动生成）
   └─ Markdown可跳转目录
   └─ 清晰展示文章结构

📍 第一章：问题与背景（1000-1500字）
   └─ 1.1 现状分析（300-500字）
   └─ 1.2 核心痛点（300-500字）
   └─ 1.3 为什么重要（300-500字）

📍 第二章：核心方法/框架（2000-2500字）⭐ 核心部分
   └─ 2.1 核心概念定义（500-700字）
   └─ 2.2 方法论完整拆解（800-1000字）
   └─ 2.3 关键要素解析（700-800字）

📍 第三章：实操步骤与案例（2000-2500字）⭐ 核心部分
   └─ 3.1 步骤一：[具体步骤]（400-500字）
   └─ 3.2 步骤二：[具体步骤]（400-500字）
   └─ 3.3 步骤三：[具体步骤]（400-500字）
   └─ 3.4 实战案例深度复盘（600-800字）

📍 第四章：进阶技巧与FAQ（1000-1500字）
   └─ 4.1 进阶技巧（3-5个）（400-500字）
   └─ 4.2 常见误区（3-5个）（300-500字）
   └─ 4.3 FAQ问答（5-8个）（400-600字）

📍 第五章：总结与行动建议（300-500字）
   └─ 全文核心要点回顾
   └─ 立即可执行的3个行动
   └─ 下一步建议

📍 CTA引导（100字）
```

### 总字数分配

- 第二章：2000-2500字（28-30%）⭐

- 第四章：1000-1500字（14-17%）
- 第五章：300-500字（4-6%）

## 框架选择流程（P1自动判定）

```
选题分析 → 判定内容类型 → 选择框架

关键词匹配规则：
- 方法/SOP/原理/系统 → 深度干货
- 工具/教程/步骤/实测 → 工具实操
- 经历/踩坑/复盘/感悟 → 个人故事
- 模板/清单/资料/合集 → 素材分享
- 热点/事件/趋势/新闻 → 热点蹭流
- 完整/指南/白皮书/方法论 → 深度长文
```

---

## 使用注意事项

1. **字数分配严格遵守**：不可随意增减，影响内容节奏
2. **框架可微调**：根据实际内容调整，但核心要素不可缺
3. **验证机制**：Step 4实时验证大纲是否符合框架（P2流程）
4. **账号适配**：不同账号使用同一框架时，需调整语言风格
5. **CTA统一**：所有框架的CTA都使用统一的"诊断"模板
