# 公众号内容生成器 - 智能体使用指南 v1.1

## 项目概述
本智能体组用于为「IP内容工厂」4个公众号矩阵批量生成高质量长文内容（1500-3000字），基于李可IP人设，支持5种内容类型框架。

**v1.1 核心功能**：
- P1 选题分析与框架选择
- P2 公众号长文生成（含标题、正文、CTA）
- P3 质量检测与优化（AI痕迹检测、口语化优化）
- P4 配图提示词生成（封面图+小节配图）
- 支持4号矩阵差异化输出
- 内置李可IP人设锚定
- AI痕迹规避机制

## 智能体工作流程

### 业务执行步骤

**P1 - 选题分析与框架选择**
- 分析选题的核心价值和目标受众
- 判定内容类型（深度干货/工具实操/个人故事/素材分享/热点蹭流）
- 选择对应的写作框架
- 推荐发布账号和CTA钩子

**P2 - 公众号长文生成**
- 基于框架结构生成1500-3000字正文
- **必须包含**：李可IP人设和语言风格
- **必须包含**：开头钩子（3秒抓住注意力）
- **必须包含**：文末CTA引导
- 输出完整格式（标题+备选+摘要+正文+CTA+发布建议）

**P3 - 质量检测与优化**
- AI痕迹检测（A级禁用词检查）
- 口语化风格检查
- 人设一致性验证
- 输出优化建议和修改后版本

**P4 - 配图提示词生成**
- 从文章提取核心信息生成配图提示词
- 封面图：900×383px（头条）/ 500×500px（次条）
- 小节配图：1080×720px，每个核心小节1张
- 支持多种风格：黑板报风格、科技风格、插画风格等
- 输出可直接用于Midjourney/DALL-E/SD的完整提示词

## 核心技巧（从公众号爆款内容提炼）

### 1. 开头3秒抓住注意力
```
❌ 错误：今天来聊聊AI写作这个话题...
✅ 正确：90%的人用AI写内容，都卡在同一个地方。
```

### 2. 像说话一样写作
```
❌ 书面腔：综上所述，AI工具对于提升效率具有显著作用。
✅ 口语化：说白了，用对了AI，效率能提10倍不止。
```

### 3. 数据和案例支撑
```
❌ 空洞：AI能大大提升效率
✅ 具体：我用AI写脚本，从4小时压缩到10分钟，一周能产出50条
```

### 4. 段落节奏控制
```
- 每段不超过4行
- 重要观点单独成段
- 用空行制造呼吸感
```

### 5. AI痕迹规避（A级禁用词）
```
❌ 禁用：「这不是X，是Y」「更扎心的是...」「你知道吗？」
✅ 删除或用自然表达替代
```

## 注意事项

1. **字数控制**：公众号长文1500-3000字为佳
2. **段落节奏**：每段不超过4行，用空行制造呼吸感
3. **标题规范**：15-25字，必须含数字，直击痛点
4. **AI痕迹**：生成后务必执行P3检测，删除A级禁用词
5. **人设一致**：所有内容必须符合李可IP人设
6. **CTA必须**：每篇文章必须有文末CTA引导
7. **差异化**：4号发布不同内容，避免重复
8. **发布时间**：建议早7-9点、中午12-13点、晚20-22点
