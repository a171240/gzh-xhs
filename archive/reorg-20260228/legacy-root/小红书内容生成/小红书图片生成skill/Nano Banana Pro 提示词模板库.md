## 四、文字渲染优化技巧

  

### 1. 减少文字量

- 主标题：6-8字最佳

- 副标题：10-15字

- 总字数不超过25字

  

### 2. 明确字体描述

```

✅ 好的描述：

"圆润的现代无衬线黑体，字重为Bold，类似苹方-简 粗体的感觉"

  

❌ 模糊描述：

"好看的字体"

```

  

### 3. 精确颜色指定

```

✅ 好的描述：

"深棕色，类似摩卡咖啡色（#5D4037）"

  

❌ 模糊描述：

"棕色"

```

  

### 4. 位置和大小

```

✅ 好的描述：

"文字位于画面上方1/4处，水平居中，字号占画面宽度的70%"

  

❌ 模糊描述：

"文字在上面，大一点"

```

  

### 5. 处理渲染失败

如果文字渲染不准确，尝试：

- 减少文字数量

- 使用更简单的字体描述（如"简洁的黑体"）

- 分开主副标题，分两次生成

- 改用英文版提示词生成图片，后期用Canva加文字
