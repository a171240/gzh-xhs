## 八、关键词SEO布局规范

### 标题布局
- **必须**：包含1个主关键词
- **建议**：自然融入，不堆砌
- **字数**：≤20字

### 正文布局
- **首段**：主关键词 + 1-2个长尾词
- **小标题**：使用长尾词
- **频率**：自然出现，不刻意重复

### 话题标签布局
- **数量**：10个
- **构成**：3个核心词 + 7个长尾词
- **格式**：#关键词

### 示例
```
标题：AI选题工具做不起来？90%的人卡在选题库

首段：很多人用了AI选题工具还是不会选题，问题出在哪？
我测试了20多个内容排期工具，发现90%的人卡在同一个地方：没有选题库。

话题：#AI选题工具 #内容排期 #选题库 #内容日历 #自媒体运营 #小红书运营 #脚本生成 #内容效率 #团队协作 #返工
```

## 使用说明

1. **选择账号**：根据账号定位选择对应关键词库
2. **匹配象限**：高热高转化 > 低热高转化 > 高热低转化
3. **写入标题**：主关键词自然融入标题
4. **布局正文**：首段和小标题使用长尾词
5. **设置话题**：10个话题标签覆盖核心词和长尾词
6. **投放定向**：根据阶段选择定向关键词
