# 信息图模板库 - 5种类型详细规范

> 本文件为小红书Skill的P1封面信息图模板，使用Nano Banana Pro格式生成高密度数据可视化信息图。
> 核心要求：**一张图 = 一篇2000字文章的完整信息量**

---

## 一、通用规范

### 信息密度要求
- **主标题**：1个（8-15字）
- **核心数据**：2个（超大字号高亮）
- **核心模块**：4-6个
- **每模块子要点**：2-4个
- **关键数据**：5-8个（百分比、数字、对比值）
- **关系线**：连接各模块的逻辑关系
- **总信息点**：**20-30个**

### 视觉规范
- **画幅**：3:4竖版（1080×1440px）
- **留白**：30-40%（信息充实但有序）
- **背景**：浅色渐变（有层次感）
- **字体**：无衬线粗体（标题）+ 常规体（正文）+ 数字专用（数据）

### 账号配色

| 账号 | 主色 | 辅助色 | 背景色 | 数据高亮 |
|------|------|--------|--------|----------|
| A号（转化） | 紫色#7C3AED | 浅紫#A78BFA | 浅灰#F8F8F8 | 紫色渐变 |
| B号（交付） | 灰色#6B7280 | 浅灰#9CA3AF | 白色#FFFFFF | 深灰高亮 |
| C号（观点） | 深灰#374151 | 中灰#6B7280 | 米白#FAFAFA | 黑色高亮 |

### 通用负面提示词
```
文字变形扭曲、文字模糊、信息杂乱无序、层次不清、大面积空白、
人物照片、英文字母、装饰性图案过多、卡通风格、手绘风格
```

## 七、类型选择指南

### 根据内容快速选择

| 你的内容是关于... | 推荐类型 | 关键词 |
|------------------|----------|--------|
| 怎么做、步骤、流程 | **流程型** | SOP、教程、指南、步骤 |
| 什么是、体系、框架 | **层级型** | 模型、体系、分类、架构 |
| 哪个好、选择、评测 | **对比型** | 对比、评测、选择、差异 |
| 为什么、原因、机制 | **因果型** | 原因、本质、机制、解决 |
| 如何选、优先级、策略 | **矩阵型** | 策略、优先级、分类、象限 |

### 账号×类型推荐组合

| 账号 | 常用类型 | 原因 |
|------|----------|------|
| A号（转化） | 对比型、流程型 | 展示方案优势，引导决策 |
| B号（交付） | 流程型、层级型 | 展示专业SOP，建立信任 |
| C号（观点） | 因果型、矩阵型 | 展示深度思考，筛选人群 |

## 八、使用说明

### 工作流程

1. **确定内容主题**：明确要讲什么
2. **生成完整内容**：先写2000字级别的结构化内容
3. **选择信息图类型**：根据内容性质选择
4. **填充模板变量**：将内容填入对应模板
5. **生成图片**：使用Gemini执行提示词
6. **质检验证**：检查信息点数量是否达到20-30个

### 质检标准

- [ ] 信息点数量：20-30个
- [ ] 核心数据：2个以上，大字号高亮
- [ ] 模块数量：4-6个
- [ ] 关系线：清晰呈现模块间逻辑
- [ ] 配色正确：与账号匹配
- [ ] 角标正确：显示账号标识
- [ ] 文字清晰：手机端可读
