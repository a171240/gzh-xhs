## 三、4号矩阵策略（统一IP人设）

### 核心策略：1个IP人设 + 4个号分散风险

**为什么用同一个IP人设？**
- 降低人设维护成本
- 强化IP认知度
- 内容可复用和互相引流
- 测试哪个号能跑出来

**4号差异化策略**

| 维度 | 说明 |
|------|------|
| 人设 | **完全一致**：同一个人、同一个故事、同一个风格 |
| 账号名称 | **相似但不同**：XX说AI内容 / XX的AI笔记 / AI内容XX / XX聊AI写作 |
| 内容 | **同主题不同文章**：同一选题方向，但不发相同的文章 |
| 发布时间 | **错开发布**：避免同时段发布，分散测试最佳时间 |
| 引流钩子 | **统一**：都导向同一个私域/产品 |

### 内容分发逻辑

```
选题池（每周20-30个选题）
    ↓
生成28篇内容（每号7篇）
    ↓
按账号特点微调后发布
```

**4号内容分配原则：**
- 每周总产出28篇内容
- 每篇内容只发1个号（不重复发布）
- 4个号覆盖不同的选题角度
- 跑得好的内容可以在其他号"二创"

### 每号每周内容配比

| 内容类型 | 数量 | 说明 |
|---------|------|------|
| 深度干货 | 2篇 | AI写作方法、批量生产SOP |
| 工具实操 | 2篇 | AI工具测评、使用技巧 |
| 个人故事 | 1篇 | 踩坑复盘、真实经历 |
| 素材分享 | 1篇 | 提示词、标题模板、金句 |
| 热点蹭流量 | 1篇 | 结合热点的AI内容话题 |
| **周发布量** | **7篇** | - |

## 五、内容生产SOP

### 日常工作流程（全职4小时+）

| 时间段 | 任务 | 工具/方法 |
|--------|------|----------|
| 08:00-09:00 | 热点扫描 + 选题确定 | 微信指数、新榜、热点引流选题智能体 |
| 09:00-11:00 | 内容生产（4篇） | P7选题库 → P8脚本创作 → P9口语化优化 |
| 11:00-12:00 | 内容质检 + 排版 | 脚本原创度检测、AI痕迹去除 |
| 14:00-15:00 | 发布 + 数据监控 | 4号轮流发布，记录数据 |
| 15:00-16:00 | 复盘 + 素材库更新 | 分析5指标，更新爆文模板库 |
| 20:00-21:00 | 评论互动 + 私域引导 | 回复评论，引导加微信 |

### 内容生产详细SOP

**Step 1：选题（使用P7选题库生成大师）**
- 输入：行业+热点+IP定位
- 输出：50个热点选题 + 20个故事选题 + 80个情绪选题
- 筛选标准：流量潜力 > 品牌安全 > 转化价值

**Step 2：脚本创作（使用P8脚本创作中心）**
- 根据选题类型选择对应智能体：
  - 引流类 → 深度共鸣/金句型/奇葩问题
  - 理性类 → 人生故事
  - 产品类 → 促销钩子/产品展示

**Step 3：口语化优化（使用P9口语化大师）**
- 去除AI味（A级禁用词必删）
- 增加细节和逻辑连贯性
- 植入个人经历/观点

**Step 4：原创度检测**
- 使用「脚本原创度双盲测试」
- 确保重复率 < 15%
- 跨多篇合成，避免单篇仿写

## 九、风险控制

### 合规红线

1. **内容原创度**：单篇重复率 < 15%，跨多篇合成
2. **引用规范**：观点引用+自己解释，不整篇照搬
3. **敏感话题**：避免涉政、涉争议热点
4. **健康/财经/法律**：必须做事实核查

### 账号安全

1. **一号一IP**：已配置，严格执行
2. **内容差异化**：4号内容不完全相同，避免互相打架
3. **发布节奏**：新号每天1-2篇，逐步加量
4. **养号周期**：首月重质量，不急于堆量
