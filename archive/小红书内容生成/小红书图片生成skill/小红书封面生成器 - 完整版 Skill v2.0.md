## P1 - 文案分析与视觉规划

  

### 任务

分析用户提供的小红书文案，提取核心元素并规划视觉方向。

  

### 分析框架

  

#### 1. 内容解构

- **核心卖点**：文案要传递的核心价值是什么？

- **情绪价值**：想要唤起用户什么情感？

- **目标受众**：这篇内容写给谁看？

  

#### 2. 关键词提取

- **主题词**：文案的核心主题（1-2个）

- **场景词**：适合的视觉场景（2-3个）

- **情绪词**：画面应传递的情绪（1-2个）

- **风格词**：适合的视觉风格（1-2个）

  

#### 3. 视觉方向建议

根据内容类型推荐封面类型：

- 产品展示型（种草、测评）

- 人物场景型（穿搭、妆容、生活方式）

- 纯文字海报型（干货、观点、情感）

- 信息图卡片型（教程、对比、清单）

- 手账贴纸型（OOTD、生活方式）

- 氛围意境型（旅行、美食、情绪）

- 3D信息图型（食谱教程、步骤讲解）

  

### 输出格式

```markdown

### 1. 内容解构

- 核心卖点：[xxx]

- 情绪价值：[xxx]

- 目标受众：[xxx]

  

### 2. 关键词矩阵

| 类别 | 提取结果 |

|------|----------|

| 主题词 | xxx |

- 推荐封面类型：[xxx]

- 推荐构图：[xxx]

```

## P2 - Nano Banana Pro 提示词生成

  

### 核心原则

  

#### 原则一：比例声明必须放最前面

```

画幅比例3:4竖版。  ← 第一句话！

```

  

#### 原则二：描绘场景，而非堆砌关键词

❌ 错误：`女生，护肤品，ins风，粉色，好看`

✅ 正确：`一位年轻女性手持精致的护肤瓶，坐在充满阳光的梳妆台前，柔和的晨光透过白色纱帘洒落...`

  

#### 原则三：材质描述要极致精确

❌ 模糊：`手绘风格背景`

✅ 精确：`在一张有纹理的、陈旧的羊皮纸上的水彩和钢笔淡彩插画`

  

#### 原则四：中文字体用专业描述

| 字体类型 | 专业描述方式 | 适用场景 |

|---------|-------------|----------|

| 隶书 | 雄浑大气的隶书，竖排书写 | 古典标题、传统文化 |

| 行书 | 优美的、略带不完美感的手写书法风格（行楷） | 手写感、温馨氛围 |

| 草书 | 红黑相间、对比鲜明的大号毛笔草书字体 | 艺术海报、情绪表达 |

| 小篆 | 优美纤细、典雅的小篆字体 | 印章、古风元素 |

| 圆润黑体 | 圆润的现代无衬线黑体，加粗 | 现代标题、简约风 |

| 手写体 | 略带倾斜的俏皮手写风格 | 手账、少女风 |

  

#### 原则五：必须添加负面提示词

```

负面提示词：模糊, 水印, 乱码, 畸形, 低质量

```

  

**负面提示词库**：

| 场景类型 | 负面提示词 |

|---------|-----------|

| 通用 | 模糊, 畸形, 低质量, 水印, 乱码 |

| 手绘/插画 | 3D, 照片感, 写实, 塑料质感 |

| 古风/传统 | 英文, 罗马字母, 现代字体, 横排文字 |

| 信息图 | 杂乱, 空白过多, 电脑字体 |

| 人物场景 | 人物变形, 多余肢体, 面部扭曲 |

  

### 输出格式

  

必须输出：

1. **完整提示词（中文版）** - 200-400字

2. **完整提示词（英文版）** - 150-300 words

3. **提示词结构解析** - 各部分作用说明

4. **变体建议** - 2-3个可替换元素

## 注意事项

  

1. **提示词语言**：英文提示词效果更稳定，建议优先使用英文版

2. **文字渲染**：中文文字建议控制在10字以内，确保清晰度

3. **尺寸比例**：小红书封面推荐3:4竖版，信息图推荐16:9横版

4. **风格一致性**：同一账号建议保持统一的视觉风格

5. **负面提示词**：根据场景类型选择对应的负面提示词组合
