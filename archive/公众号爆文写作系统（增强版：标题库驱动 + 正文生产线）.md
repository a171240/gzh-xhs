
> 用途：把这份文档直接喂给AI，让它“先出标题 → 再出大纲 → 再出正文 → 最后自检”，稳定产出高点击/高完读/可转发/可转化的公众号文章。  
> 重点：别再只学“结构”。你已经有结构了；真正拉开差距的是**标题工程化**与**节奏/证据/反驳/传播点/转化链路**。

## 0. 快速开始（你只要填一张 Brief）
### 0.1 给AI的输入 Brief（固定格式）
- 主题矿区：职场/失业｜存款/赚钱｜35岁/中年｜不上班/副业｜家庭/父母｜人际/社交｜认知/人性｜消费/房产
- 目标人群：例如 90后/35岁打工人/宝妈/普通人/县城青年
- 核心矛盾：一句话（“卡在哪里 + 代价是什么”）
- 场景证据：一个具体画面（账单/通知/饭局/体检/房贷日/相亲局）
- 你希望读者做的第一步：一个最小动作
- 引流物：清单/表格/模板/案例库（文末领取）
- 禁区：不要碰的敏感点（可选）

### 0.2 生产流程（AI必须按顺序）
1) 用“标题系统”先生成 20 个标题候选 → 打分 → 选 Top2  
2) 用“正文系统”按 Top2 各写 1 份大纲（01/02/03）  
3) 选择最强大纲写全文  
4) 按“发布前检查清单”自检，不通过就重写

## 1. 标题系统（来自 800+爆款标题的可复用结构）
> 数据来源：3个爆款标题文档，共 800+ 标题；更新日期 2026-01-06；并用 🔥🔥🔥（重复≥3）/🔥🔥（重复=2）标记验证强度。:contentReference[oaicite:0]{index=0}  
> 结论：标题不是灵感，是工程。重复出现的标题意味着：**选题方向被验证 + 公式有效 + 情绪触发精准**。:contentReference[oaicite:1]{index=1}

### 1.1 先盯“重复爆款区”：这是你最值钱的矿
**🔥🔥🔥 超级爆款（重复≥3）**呈现的稳定模式（举例）：
- 时间/年份 + 预测 + 悬念（“2026年春节即将出现的反常现象”）  
- 条件判断句 + 一锤定音（“你公司烂不烂放个假就知道了”）  
- 身份限定 + “主线任务/唯一出路”（“没有父母托举的普通人一生只有一个主线任务”）  
- 很多人 + 没意识到 + 红利/真相（“很多人根本没有意识到这个时代最大的红利”）  
- 群体 + 状态崩溃（“大批打工人快撑不住了”）  
- 因果 + 代价残酷（“父母的长寿必然来自一位子女的牺牲”）  
- 反常识反转（“失业或辞职反而是好事”）  
- 现象 + 答案（“成年人的绝交朋友圈的横杠就是答案”）  
- 知识名词 + 立刻行动（“儿童心智…立刻马上远离”）  
以上都属于“多平台验证”的钩子结构。:contentReference[oaicite:2]{index=2}

> 用法：你写任何赛道，都先把内容套进这些**已验证结构**，别自创花活。

### 1.4 标题积木（把标题拆成可拼装组件）
- 人群标签：`90后/35岁/普通人/宝妈/底层家庭/打工人`
- 场景词：`年终/绩效季/体检/过年回家/房贷日/相亲局`
- 冲突词：`撑不住/被边缘化/发不出工资/存款归零/关系破裂`
- 结构钩子：`发现/真相/答案/反常识/千万不要/只要中一个`
- 证据锚点：`3000工资/20万存款/1200万毕业生（数字越具体越好）`

### 1.5 标题生产线（必须执行，不要凭感觉挑）
**步骤A：先产 20 个候选**  
要求：至少覆盖 5 种不同公式（Top10里挑），避免同质化。

**步骤B：打分（每条25分，选Top2）**
1) 对号入座（人群清晰）0-5  
2) 信息缺口（想点开）0-5  
3) 情绪强度（焦虑/好奇/警示）0-5  
4) 可信锚点（数字/场景）0-5  
5) 新鲜度（不撞原句）0-5  

**步骤C：去重改编（4个旋钮，拧一个就变新）**
1) 换人群：90后→宝妈/县城青年  
2) 换场景：发工资→绩效季/过年回家  
3) 换数字：3→4/5；20万→10万/50万  
4) 换力度：建议→千万别/立刻/别再

### 1.6 标题优化技巧（写作层面的硬规则）
- 去标点：让标题像一句话顺着说出来  
- 加数字：具体数字比模糊表达更有说服力  
- 加身份：让目标人群一眼认出是在说自己  
- 加情绪词：增强吸引力

## 2. 正文系统（高完读：节奏 + 证据 + 反驳 + 传播）
> 文章不是“讲道理”，是“带读者走一条情绪路径”：共鸣→解释→出口→行动。

### 2.1 正文漏斗（必须遵守）
**焦虑/冲突吸引注意 → 故事细节建立信任 → 逻辑拆解说服大脑 → 行动方案给出口 → CTA完成转化**

### 2.2 文章结构（固定四段）
1) 标题（Top2备选）  
2) 开头（黄金3秒：立人设+造场景+戳痛点+抛钩子）  
3) 正文（01/02/03清单体，每点闭环）  
4) 结尾（升华 + CTA，不要散）

### 2.3 开头四件套（写不好就别往下写）
1) 身份锚点（1句）：我是谁/我观察谁  
2) 场景细节（2-3句）：具体到动作/账单/对话  
3) 扎心矛盾（1句）：努力也没用/越省越穷/越忍越崩  
4) 钩子承诺（1句）：我拆成3点/给你一条路/最后送清单

### 2.4 每个分点必须“闭环”（确保读者不掉）
对每个 01/02/03，都按这个模块写：
- 观点（狠一句）
- 案例（细节+数字+对话）
- 底层逻辑（为什么）
- 行动第一步（怎么做，先做什么）
- 反驳安全阀（这条不适用于谁？）
- 金句（可截图转发，**加粗**）

### 2.5 节奏与排版（完读率生死线）
- 每2-4行强制换行（手机阅读友好）
- 每一屏必须出现至少一个“爽点”：反转/金句/细节/提问/结论
- 单段不超过 120-180 字（超过就拆）

### 2.6 传播点设计（至少3句可截图金句）
金句类型：
1) 朋友圈状态句（短、狠、像判词）  
2) 行动口令句（可执行、像命令）  
3) 身份宣言句（读者愿意认领）

## 3. 转化系统（别只在结尾喊关注）
### 3.1 全篇三处埋点（建议固定）
- 开头轻埋：“最后我给你一张清单/模板”  
- 正文中埋：“第一步我写成了表格（文末领）”  
- 结尾强CTA：“评论关键词/关注下一篇/加微信领资料”

### 3.2 CTA写法（低门槛 + 高确定性）
- “评论区回【关键词】我发你清单/表格/模板”
- “点个【推荐】，我把案例库放出来”
- “关注我，下一篇写：XX（承诺必须具体）”

## 5. AI提示词库（复制即用）
### 5.1 标题生成 Prompt
> 你是爆款标题编辑。基于我的Brief，用Top10公式至少覆盖5种结构生成20个标题；每条标注【公式标签】并按25分制评分；最后选Top2，并说明每条1句话为什么更可能爆。  
> 硬约束：标题18-28字优先；必须命中“人群/场景/数字/反差/强动词”至少3项；避免与已知爆款原句完全重复；尽量去标点。

### 5.2 大纲生成 Prompt
> 基于Top2标题，各输出一份文章大纲：开头四件套 + 01/02/03（每点给：观点/案例细节/底层逻辑/第一步/金句）。要求：每点都要能落到现实细节与数字。

### 5.3 全文生成 Prompt
> 选择其中最强大纲写全文：短段落强换行；每屏一个爽点；至少3句可截图金句（加粗）；至少1处反驳安全阀；全篇三处埋点；结尾给明确CTA（关键词领取）。写完附自检勾选表。
