## 使用说明

### 1. 查询金句
每次生成标题/开头前，优先检索本库：
```
1. 从Brief的"主题矿区"提取核心词
2. 在表格中搜索相关痛点
3. 若找到，使用"公众号标题改写"或"公众号开头改写"
4. 标注来源，避免重复使用同一条金句
```

### 2. 改写规则
- **仅作为灵感来源**：必须进行改写，不能原文照搬
- **保留核心痛点**：改写时保留原评论的核心痛点和情绪
- **适配公众号风格**：改写成符合公众号风格的标题或开头

### 3. 录入规则
发现高价值抖音评论时，按以下格式录入：
```
- 原评论：原文（不改动）
- 痛点提炼：提炼出的用户痛点
- 公众号标题改写：适配公众号的标题改写
- 公众号开头改写：适配公众号的开头改写
- 来源视频：原视频链接（用于溯源）
- 录入日期：YYYY-MM-DD
```

### 4. 更新频率
- 每周录入5-10条高价值评论
- 每月清理使用次数<3的评论
