## 爆款类型分类

请选择最符合的类型（单选）：

- [ ] **系统方法论型**：案例驱动 → N层递进 → 可复用框架
  - 特征：有明确的步骤、框架、系统
  - 示例："普通人逆袭三步法"

- [ ] **心理学洞察型**：现象 → 根源分析 → 心理陷阱 → 解决方案
  - 特征：深挖心理机制、揭示底层原因
  - 示例："只要你没在执行，你就是在逃避执行"

- [ ] **反常识观点型**：观点 → 案例 → 底层逻辑 → 应用
  - 特征：颠覆常规认知、提出反直觉观点
  - 示例："这个世界是可以作弊的"

- [ ] **行动清单型**：目标 → N个行动 → 每个行动的细节 → 组合效应
  - 特征：可马上执行、低成本高频、有反馈循环
  - 示例："三个实现财富自由的基础行动"
