# Skill: 短视频脚本生产

## 触发
用户说"写短视频：{Brief}"

## Brief 输入模板
- 平台：抖音 / 视频号
- 选题：
- 目标人群：
- 时长：60秒 / 90秒 / 120秒 / 180秒
- 风格：讲解型 / 故事型 / 数据型 / 演示型
- 核心矛盾：
- CTA：

## 动作

### Step 1: 素材检索
按"深化选题"Skill 的检索规则，先检索素材库。

### Step 2: 脚本结构
```
## 黄金3秒钩子
{开头钩子 - 必须在3秒内抓住注意力}

## 正文
### 段落1：问题/痛点（15-30秒）
{建立共鸣}

### 段落2：核心观点/方法（30-60秒）
{信息增量，每段有一个记忆点}

### 段落3：案例/证据（15-30秒）
{增强说服力}

## CTA（5-10秒）
{明确的行动号召}

## 字幕关键帧
{需要强调的文字，用于字幕/贴纸}
```

### Step 3: 输出
保存到 `02-内容生产/{平台}/生成内容/{日期}/{账号}-{YYYYMMDD}-{短标题}.md`

文件包含：
- YAML frontmatter（日期、平台、选题、时长、风格）
- 完整脚本
- 字幕关键帧
- 配乐建议（可选）

## 规则
- 每句话不超过15字（口语节奏）
- 前3秒必须有钩子
- 正文每15-20秒一个信息转折点
- CTA 必须明确且与引流物一致
- 不用书面语，用口语化表达
