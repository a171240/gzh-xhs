# Skill: 对标文案分析

## 触发方式
用户说"分析对标文案：{对标文案标题或路径}"

**输入示例**：
```
分析对标文案：普通人逆袭三步法
```

或

```
分析对标文案：03-素材库/对标链接库/提取正文/2026-03-05/分享一条超级干货-普通人逆袭三步法-2026-03-05.md
```

---

## 角色定义
你是"爆款逻辑分析师"：
- 精通内容结构分析和情绪触发点识别
- 能够从完整文案中提炼可复用元素
- 擅长将对标文案的爆款逻辑转化为可执行的改写建议

---

## 执行流程

### Step 1: 读取对标文案

1. 若用户提供完整路径，直接读取
2. 若用户仅提供标题，在 `03-素材库/对标链接库/提取正文/` 目录下搜索匹配的文件
3. 读取文案的完整内容

**输出**：
- 文案标题：____
- 文案路径：____
- 字数：____
- 提取日期：____

---

### Step 2: 按照分析模板逐项分析

参考 `03-素材库/对标链接库/分析模板.md`，逐项完成以下分析：

#### 2.1 爆款类型分类
- 判断文案属于哪种类型（系统方法论型/心理学洞察型/反常识观点型/行动清单型）
- 说明判断依据

#### 2.2 结构模板提炼
- 开头：钩子类型、钩子内容、情绪触发点
- 正文：分几个部分，每部分的核心观点
- 结尾：收口方式、CTA类型

#### 2.3 情绪触发点分析
- 主要情绪、次要情绪
- 触发场景、痛点描述
- 触发时机（开头/中段/结尾）

#### 2.4 可复用元素提炼
- 金句提炼（3-5条）：标注用途、潜力标签、适用主题
- 核心观点（1-3个）：支撑案例、底层逻辑、可改写方向
- 案例素材（1-3个）：人物、场景、结果、可替换性
- 框架模板：框架名称、适用场景、结构、可复用性

#### 2.5 适配平台分析
- 公众号：是否适合，原因，改写建议
- 小红书：是否适合，原因，改写建议
- 抖音/视频号：是否适合，原因，改写建议

#### 2.6 公众号改写建议
- gongchang（系统方法论）：改写方向、核心逻辑、标题建议、开头建议
- ipgc（情绪共鸣）：改写方向、核心逻辑、标题建议、开头建议
- zengzhang（数据增长）：改写方向、核心逻辑、标题建议、开头建议
- shizhan（轻实操）：改写方向、核心逻辑、标题建议、开头建议

#### 2.7 三层表达分析（富贵方法论）
- 事实层：主要事实、出现位置
- 感受层：主要感受、出现位置
- 想象层：主要想象、出现位置
- 三层转译示例

---

### Step 3: 输出分析报告

生成完整的分析报告，保存到：
`03-素材库/对标链接库/分析报告/YYYY-MM-DD/{标题}-分析.md`

**文件命名规则**：
- 日期：对标文案的提取日期
- 标题：对标文案的标题（简化版，去除特殊字符）

**报告结构**：
- 使用分析模板的完整结构
- 所有字段必须填写完整
- 分析总结部分必须给出明确的建议用途

---

### Step 4: 更新素材库（可选）

根据分析结果，询问用户是否需要：

1. **将提炼的金句写入金句库**
   - 目标文件：`03-素材库/金句库/{主题}.md`
   - 标注来源：对标文案
   - 标注原文标题和提取日期

2. **将提炼的框架写入内容框架库**
   - 目标文件：`03-素材库/内容框架库/爆款结构模板.md`
   - 新增框架模板

3. **将提炼的选题写入选题池**
   - 目标文件：`01-选题管理/选题规划/金句选题池.md`
   - 标注来源：对标文案

## 分析要点

### 爆款类型判断标准

**系统方法论型**：
- 有明确的步骤（如"三步法"、"五个阶段"）
- 有可复用的框架或系统
- 案例驱动，每步都有具体案例
- 强调可操作性和可复制性

**心理学洞察型**：
- 深挖心理机制，揭示底层原因
- 有"现象→根源分析→心理陷阱→解决方案"的逻辑链条
- 强调认知升级和思维转变
- 常用心理学术语或理论

**反常识观点型**：
- 颠覆常规认知，提出反直觉观点
- 有"观点→案例→底层逻辑→应用"的结构
- 标题或开头就抛出反常识观点
- 强调"其实"、"真相是"、"你以为...其实..."

**行动清单型**：
- 有明确的N个行动（如"三个行动"、"五件事"）
- 每个行动都有具体的细节和方法
- 强调可马上执行、低成本高频
- 有反馈循环或组合效应

### 金句提炼标准

**开头钩子**：
- 能直接用作文章开头
- 有强烈的情绪触发或好奇心触发
- 示例："90%的人学了100个技巧，还是做不起来"

**观点支撑**：
- 能作为正文的核心观点
- 有一定的深度和洞察力
- 示例："复制不是创新，是普通人的起点"

**反驳安全阀**：
- 能作为风控补充，预防杠精
- 有"但是"、"当然"、"不过"等转折
- 示例："当然，这不是说创新不重要，而是说普通人一开始不适合创新"

**行动建议**：
- 能作为CTA承接
- 有明确的行动指引
- 示例："从今天开始，找一个你能够得着的人，像素级复制他"

### 公众号改写建议要点

**gongchang（系统方法论）**：
- 强调框架、机制、对比、底层逻辑
- 标题公式：很多人+现象 / 系统拆解 / 机制反转
- 开头：数据钩子 + 系统性问题
- 正文：框架展开 + 每步有逻辑

**ipgc（情绪共鸣）**：
- 强调人生阶段、关系张力、情绪反转
- 标题公式：年龄+状态+反转 / 关系张力 / 共鸣切口
- 开头：场景钩子 + 情感共鸣
- 正文：故事展开 + 情绪触发

**zengzhang（数据增长）**：
- 强调指标语言、样本洞察、增长杠杆
- 标题公式：数字+反差 / 样本洞察 / 增长结论
- 开头：数据钩子 + 增长案例
- 正文：数据支撑 + 增长逻辑

**shizhan（轻实操）**：
- 强调避坑、清单、可马上执行
- 标题公式：避坑清单 / 行动步骤 / 轻反常识
- 开头：避坑钩子 + 实操场景
- 正文：清单展开 + 每步可执行
