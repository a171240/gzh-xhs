# 美容内容底座｜运行总纲

定位：给生活美容 / 皮肤管理 / 面部护理 / SPA 放松型门店提供可复用的运行时规则。

适用范围：
- 小红书 / 短视频内容生成。
- 以生活美容门店为主，不默认扩到轻医美、产后修复、盆底、身体管理。

一句话链路：

`关键词意图 -> 具体人 -> 防御心态 -> 向往画面 -> 门店规矩 -> XHS短视频输出`

## 核心判断

- 顾客不是按年龄进入美容院的，而是按 `问题 + 场景 + 防御心态` 进入的。
- 年龄不是前台搜索入口，但仍然是后台翻译维度。
- 搜索时，顾客冲着效果和状态来；留下来，取决于服务过程中有没有不再被消耗。
- 美容院前台不能只卖项目名，要卖“状态答案”。
- `坑点是入口，向往是成交理由，规矩是托底。`

## 四条前台原则

1. 先判关键词意图，不分类不出稿。
2. 先还原具体人，再决定正文刀口。
3. 先拆防御，再谈专业。
4. 先给生活感和被照顾感，再用规矩把信任托住。

## 美容消费的六个底层动机

每篇默认只选 1-2 个，不要全塞进同一篇。

- 悦己与自我修复
- 被照顾与合法休息
- 体面与关系场合
- 职场外貌劳动与状态管理
- 掌控感与确定性
- 社交验证与闺蜜影响

## 默认不做什么

- 不把公众号长文、FAQ 页、落地页、套餐菜单当成 v1 主输出。
- 不把内容写回“纯反套路吐槽号”。
- 不把内容写成“纯项目说明书”。
- 不默认承接高风险医疗、产后、盆底、身体管理类关键词。
