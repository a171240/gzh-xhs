# 美容内容底座｜关键词意图与阶段路由

## 1. 关键词意图路由

进入主流程的关键词只保留 4 类。

### 1.1 问题修复

典型词：
- 去黑头
- 毛孔粗大怎么改善
- 祛痘印
- 闭口怎么去掉
- 美白

用户表面在问：
- 这个问题怎么处理。

用户真实在买：
- 我想变干净一点、稳定一点，但不想把脸折腾得更糟。

正文重心：
- `结果入口 + 反套路承接`

默认叙事比例：
- `2-5-3`

### 1.2 信任/怀疑

典型词：
- 美容院做脸有用吗
- 美容院补水项目有用吗
- 美容院去黑头有用吗
- 美容院套路
- 美容院退费

用户表面在问：
- 值不值、靠不靠谱、会不会被坑。

用户真实在买：
- 我想做，但不想再被消耗一遍。

正文重心：
- `拆防御 + 规则先行`

默认叙事比例：
- `4-3-3`

### 1.3 放松养护

典型词：
- SPA
- 按摩
- 肩颈按摩
- 头部按摩
- 放松解压

用户表面在问：
- 哪种体验舒服、适合我、能不能让我松下来。

用户真实在买：
- 我想被照顾一下，不想再做决定，不想再紧绷。

正文重心：
- `向往画面 + 被照顾感`

默认叙事比例：
- `2-5-3`

### 1.4 本地找店

典型词：
- 吴江美容院
- 苏州美容院
- 美容院推荐

用户表面在问：
- 本地去哪家。

用户真实在买：
- 我想找一个不会白跑、不会第二次不敢来的地方。

正文重心：
- `长期感 + 本地确认 + 规则`

默认叙事比例：
- `1-5-4`

## 2. 阶段翻译模块

规则：
- 年龄不做前台标题入口。
- 年龄只在后台用于翻译“同一个问题在不同人生阶段里的真实买点”。

### 18-24
- 核心感觉：明天别翻车。
- 常见买点：见人安全感、痘痘和出油别太失控、别让我太狼狈。

### 25-29
- 核心感觉：别又黄又累。
- 常见买点：熬夜后的状态修复、通勤脸体面感、下班后终于被照顾一下。

### 30-34
- 核心感觉：别让我再试错。
- 常见买点：稳定、克制、长期感、不想再被项目和办卡拖着走。

### 35-44
- 核心感觉：别让我看起来这么疲惫。
- 常见买点：体面、状态管理、关系场合的自信感、舒服但不折腾。

### 45+
- 核心感觉：别让我一眼显垮。
- 常见买点：安全、舒服、被尊重、合理预期、不要高压教育。

### 独立高边界分支
- 产后 / 身体管理 只做独立分支，不进椿舍默认主链路。

## 3. 具体人还原规则

正式写正文前，必须先补完：

- 她是谁
- 她刚发生了什么
- 她最怕什么
- 她真正想要什么

缺任何一项，正文都会滑回空泛介绍。

## 4. 默认边界词

以下关键词不走椿舍默认主模板：

- 点痣
- 祛斑
- 产后修复
- 盆底
- 身体管理
- 艾灸
- 头疗
- 医美强对照词
- 外地词 / 脏词

处理方式：
- 先显式标记为边界词。
- 优先输出边界说明、选择标准、风险提醒。
- 不强行套用椿舍默认成交模板。
