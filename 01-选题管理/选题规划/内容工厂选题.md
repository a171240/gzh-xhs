## Step 0｜关键假设 / 反假设 / 待验证点

### 0.1 关键假设（7条）

1. 代运营团队真正购买的不是“写作能力”，而是**交付的可控性与可复制性**。
    
2. 搜索入口主要是“问题词/场景词”（被催稿、返工、排期乱），不是品牌词。
    
3. **第一价值=可交付文件包**会比“生成内容”更打动 B 端，因为它能交差、能沉淀。
    
4. 不支持退款会显著提高犹豫，需要用**买前预览 + 清晰使用路径 + 输出样例**抵消。
    
5. “5分钟出结果”能形成强钩子，但必须用**录屏/截图证据**证明，否则反噬。
    
6. 售后高频会集中在：找不到入口、没收到发货、激活失败、不会编辑/复用。
    
7. 早期证据不足时，“可生成的输出样例”能跑通冷启动，但要尽快补真实用户证据。
    

### 0.2 反假设（至少3条）

1. 他们只想要“爆款脚本”，不愿意执行流程。
    
2. 他们不会在小红书搜索这类词，而是靠社群/同行推荐。
    
3. 他们觉得 MD 文件太“技术”，反而降低使用门槛（更想要 Word/飞书）。
    

### 0.3 待验证点（10条）

1. 他们真实搜索更偏“排期/返工/质检”，还是“脚本/标题/选题”？
    
2. “MD 文件包”是否会造成使用门槛（要不要提供一键导入飞书/Notion的教程）？
    
3. 不支持退款是否会引发差评集中点？最常见后悔原因是什么？
    
4. 自动发货到达率/找回率（多少人会“收到了但找不到”）？
    
5. 5分钟出结果的“稳定性”是否在手机端同样成立？
    
6. 他们最认可的证据：文件结构清单 / 输出样例 / 真实案例 / 评价？
    
7. 团队使用时最痛的协作点：分工还是质检？
    
8. 他们升级到 199/599 的触发条件是什么（产能/客户数/返工率/交付压力）？
    
9. 替代方案中谁是最大威胁：通用对话机器人、淘宝模板包、飞书模板、同行SOP？
    
10. 哪些词会引来“白嫖党”，需要排除？

### 6.4 不建议做的词（≥20）+原因

- 爆款 / 必看 / 闭眼入：低质流量，B端反感
    
- 一夜起号 / 7天涨粉：高风险承诺，容易翻车
    
- 全自动赚钱 / 躺赚：引来投机党，高投诉
    
- 免费领取 / 0元白嫖：来的是垃圾流量
    
- 收徒 / 带你月入：割韭菜感，信任崩
    
- 最强神器 / 顶级工具：广告腔，掉信任
    
- 100%有效 / 保证提升：不可证伪，风险
    
- 代运营接单教程（泛）：意图偏“学接单”，不是“交付提效”
    
- AI写文案（泛）：竞争极大且同质化
    
- 小红书运营（泛）：太大词，打不过大号
    
- 副业 / 兼职 / 变现（泛）：人群不匹配，复购差
    
- 低价代运营（引来烂客）：损伤品牌
    
- 破解版/盗版/免费工具：风险流量
    
- 引流到私域（高风险表达）：平台风险
    
- “降本80%增效300%”（无证据）：反信任
