## A号 CTA（转化号）

**文件名**：`A-cta.png`
**CTA动作**：评论【诊断】
**主题色**：紫色 #7C3AED

### 生成提示词

```
画幅比例3:4竖版。

【图片类型】小红书轮播图CTA页
【尺寸】竖版3:4（1080×1440px）

【画面风格】
简洁行动导向设计。背景采用浅灰(#F8F8F8)到白色的柔和渐变，有细微纹理层次感。
紫色(#7C3AED)作为主色调和视觉焦点。整体专业、干净、有行动感。

【核心内容】
- CTA主文案：「评论【诊断】」
  - 位置：画面中央偏上，距顶35%
  - 字体：现代无衬线特粗体
  - 颜色：紫色(#7C3AED)
  - 大小：占画面宽度65%
  - 效果：醒目突出，文字清晰锐利

- 引导语：「我发你」
  - 位置：CTA主文案正下方
  - 字体：现代无衬线中等粗细
  - 颜色：深灰色(#333333)
  - 大小：CTA主文案的30%

- 获取项列表（3个白色卡片纵向排列）：
  - 卡片1：「✓ 8题五维评分」
  - 卡片2：「✓ 30天行动清单」
  - 卡片3：「✓ 定制化改造建议」
  - 位置：引导语下方，居中排列
  - 卡片样式：白色圆角卡片(#FFFFFF)，轻微阴影
  - 勾选图标：紫色(#7C3AED)
  - 文字：深灰色(#333333)，清晰可读

【装饰元素】
- 右上角角标「诊断」：紫色(#7C3AED)背景，白色文字
- 底部有紫色细线装饰
- 整体克制，无过多装饰

【负面提示词】
文字变形扭曲、文字模糊不清、字体过小、人物照片、英文字母、
微信号、二维码、电话号码、复杂背景、卡通风格、手绘风格
```

## 使用说明

1. **生成图片**：将上述提示词复制到Gemini，生成对应的CTA图片
2. **保存命名**：
   - A号：保存为 `A-cta.png`
   - B号：保存为 `B-cta.png`
   - C号：保存为 `C-cta.png`
3. **存放路径**：`assets/cta/` 目录下
4. **后续使用**：生成内容时，P6直接选用对应账号的CTA图片即可
