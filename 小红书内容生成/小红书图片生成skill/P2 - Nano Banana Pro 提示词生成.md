## 思考框架

  

在生成提示词前，请按以下步骤思考：

  

### 0. 首先确定画幅比例（必须放在提示词最前面）

- **小红书封面**：3:4竖版（推荐）或 9:16竖版

- **横版内容**：16:9

- **方形**：1:1

> 比例声明必须作为提示词的第一句话！

  

### 1. 确定画面类型

根据P1分析结果，选择最适合的封面类型：

- **产品展示型**：适合种草、测评类内容

- **人物场景型**：适合穿搭、妆容、生活方式类

- **纯文字海报型**：适合干货、观点、情感类

- **信息图卡片型**：适合教程、对比、清单类（手绘风格/黑板报风格）

- **手账贴纸型**：适合穿搭OOTD、生活方式类

- **氛围意境型**：适合旅行、美食、情绪类

- **3D信息图型**：适合食谱教程、步骤讲解类

  

### 2. 构建场景描述

用叙事性语言描述画面，而非堆砌关键词：

- 画面的主体是什么？

- 主体在做什么/呈现什么状态？

- 背景环境是什么？

- 光线和氛围如何？

- 整体色调和质感？

  

### 2.5 精确描述材质/媒介（新增重点）

## Nano Banana Pro 提示词核心原则

  

### 原则一：描绘场景，而非堆砌关键词

❌ 错误示例：`女生，护肤品，ins风，粉色，好看`

✅ 正确示例：`一位年轻女性手持精致的护肤瓶，坐在充满阳光的梳妆台前，柔和的晨光透过白色纱帘洒落，整个画面呈现温暖的奶油色调，营造出清新自然的护肤仪式感`

  

### 原则二：提供超乎寻常的具体细节

- 不要说"好看的衣服"，而是"一件米白色羊绒质感的oversize毛衣，袖口微微卷起"

- 不要说"漂亮的背景"，而是"浅灰色水泥质感的墙面，右侧有一盆尤加利叶的绿植点缀"

  

### 原则三：精确的文字渲染指令

```

画面底部居中位置有文字"主标题内容"

字体：现代无衬线黑体，加粗

颜色：深棕色（类似#5D4037）

大小：占画面宽度的60%

风格：简洁有力，带轻微阴影增强立体感

```

  

### 原则四：控制相机和构图

使用摄影术语：

- 镜头：35mm镜头、微距、广角

- 角度：平视、俯拍（flat lay）、仰拍、45度角

- 景深：浅景深模糊背景、全景深清晰

- 构图：居中对称、三分法、黄金分割
